Whonmywifi

Public Wi-Fi Security & Intruder Scanner

Whonmywifi evaluates public Wi-Fi security vulnerabilities to protect your data before you connect:

  • 🛡️ Traffic Hijack Diagnostics: Measures local hardware decryption speeds to identify active script hook injection or DOM inspection.
  • 📡 Route Proxy Verification: Inspects incoming edge headers to flag routing proxies or network redirection loops.
  • 🏢 Datacenter ISP Check: Validates the connection origin provider to warn if routed through datacenter nodes.

📥 Installing the Companion

To link the Whonmywifi companion browser core for advanced telemetry, please install the local extension manually:

  1. Open a new browser tab and navigate to chrome://extensions (or edge://extensions for Edge).
  2. Enable the Developer mode toggle in the top-right corner.
  3. Click Load unpacked in the top-left corner.
  4. Select the extension folder from this project's root directory.
